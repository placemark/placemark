### Placemark

The Placemark application. Detailed documentation is maintained
[in the wiki](https://github.com/placemark/placemark/wiki).

### Patterns

Type prefixes:

- C: Class
- f: Factory
- U: Utilities
- I: Interface
- s: Set

### Local SSL

Requires Tailscale with `tailscale cert` and the certs moved
to this directory

```
caddy start
```
